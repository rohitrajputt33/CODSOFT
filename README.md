# CodSoft Java Development Internship

This repository contains the Java projects completed during my Java Development Internship at CodSoft. Each task focuses on applying Object-Oriented Programming (OOP) principles, robust error handling, and logical problem-solving in Java.

## Projects Included

### 1. ATM Interface (Task 3)
A console-based ATM machine built with Object-Oriented design.
- **Features:** Simulates checking balances, safe withdrawals, and deposits.
- **Concepts Used:** Classes (`BankAccount` and `ATM`), exception handling for invalid user input, basic math logic.

### 2. Currency Converter (Task 4)
A dynamic currency conversion tool that fetches real-time data.
- **Features:** Connects to a live exchange rate API to convert user-inputted amounts between various currencies (e.g., USD to EUR).
- **Concepts Used:** `HttpURLConnection`, working with JSON responses, robust error handling for network issues.

### 3. Student Management System (Task 5)
A comprehensive CRUD (Create, Read, Update, Delete) system for managing student data.
- **Features:** Allows users to add, remove, search, and display student records. The data is saved to a local file, so it persists even after closing the program.
- **Concepts Used:** File I/O (`ObjectOutputStream`, `ObjectInputStream`), Collections (`ArrayList`), Serialization.
